# Solo Shuffle Blocker (솔로 셔플 블로커)

월드 오브 워크래프트에서 평점제 1인 조합전(솔로 셔플)을 플레이할 때, 같은 매치에 잡힌 다른 플레이어들과의 불필요한 채팅이나 불편한 대화를 사전에 차단하기 위해 만들어진 자동 차단 관리 애드온입니다.

## 🌟 주요 기능

1. **자동 임시 차단**
   * 솔로 셔플에 진입하면 자동으로 매치에 함께한 플레이어들을 내 차단 목록에 추가(임시 차단)합니다.
   * 솔로 셔플 시 가장 문제가 될 수 있는 악의적인 채팅을 미리 방지하여 쾌적하게 게임에 집중할 수 있습니다.
   * **보호 기능:** 같은 길드원이 매치에 있을 경우 길드원은 차단 대상에서 제외됩니다.
   * **영구 차단 유지:** 이미 플레이어가 수동으로 영구 차단해 놓았던 사람은 애드온의 자동 해제 대상에 포함되지 않고 영구 차단이 유지됩니다.

2. **자동 차단 해제 타이머**
   * 솔로 셔플 게임이 종료되면 지정된 대기 시간(기본 5분)을 세기 시작합니다.
   * 5분이 지나면 애드온이 이전에 임시로 차단했던 플레이어들을 모두 차단 목록에서 "자동 해제" 합니다.
   * 타이머가 돌아가는 동안 다른 솔로 셔플에 바로 진입할 경우, 매칭 전에 임시 차단됐던 유저들을 즉시 차단 해제하고 현재 매치 사람들을 새롭게 지정해 깔끔하게 관리합니다.

3. **비정상 종료 시 복구 로직 강화 (신규)**
   * 만약 게임 도중 튕기거나 타이머가 끝나기 전에 접속을 종료하여 임시 차단 목록이 풀리지 않아도 걱정하실 필요 없습니다.
   * 다음번 게임에 다시 접속했을 때 셔플 중이 아니라면, 잔여 임시 차단자들을 감지해 자동으로 5분 뒤 차단 해제 타이머를 다시 발동시킵니다. 
   * 애드온이 자체 저장소(SavedVariables)에 차단 목록을 지니고 있기 때문에 재접속 시에도 차단된 인원들을 잊어버리거나 무한정 남아있는 실수를 방지합니다.

4. **차단 목록 가득 참 대처 메커니즘 (신규)**
   * 와우의 기본 차단 목록 한도(기한)에 도달해 빈 공간이 꽉 차는 경우가 생길 수 있습니다.
   * 애드온이 셔플 진입 시 플레이어를 추가하려고 할 때 "가득 참"으로 인해 차단에 실패할 경우, 애드온이 관리 중인 차단자들 중 **"가장 예전(오래 전)에 차단했던 플레이어 6명"**을 즉각 차단 해제시켜 안정적으로 공간을 확보합니다.
   * 확보된 공간으로 새로 만난 매칭 유저들을 다시 안전하게 차단할 수 있습니다.

## ⚙️ 설정 및 사용 방법

기본적으로 별도의 조작 없이 설치만 하셔도 알아서 작동합니다. 옵션은 게임 내 인터페이스 설정에서 조작하거나 아래의 슬래시 명령어를 통해 설정창을 열 수 있습니다.

* `/ssb` - Solo Shuffle Blocker 설정 창을 엽니다.
* `/ssb unblock` - 설정된 타이머를 기다리지 않고 수동으로 현재 임시 차단된 모든 플레이어를 즉시 차단 해제합니다.

### 인터페이스 상세 설정
설정 창 내에서 다음과 같은 세팅 변경이 가능합니다.
* **차단 해제 대기 시간 설정 (1분~60분) :** 
  셔플 종료 후 플레이어의 차단을 유지할 시간을 결정합니다. 게임 후 연속 큐가 잡힐 상황을 고려해 기본값은 5분입니다.
* **전체 차단 해제 버튼 :** 현재 임시 차단된 목록을 원클릭으로 강제 초기화하여 비웁니다.
* **차단된 플레이어 목록 확인 :** 현재 애드온에 의해 임시 차단된 전체 유저 리스트를 실시간으로 직접 확인할 수 있습니다.
